五、创建 GitHub 仓库

  1. 打开 https://github.com/new
  2. 仓库名必须是 你的用户名.github.io
  3. 选择 Public
  4. 不要勾选任何初始化选项
  5. 点击 Create repository

七、开启 GitHub Pages

  1. 打开仓库的 Settings → Pages
  2. Source 选择 Deploy from a branch
  3. Branch 选择 main，文件夹选 / (root)
  4. 点击 Save

等 1~2 分钟，访问 https://你的用户名.github.io 即可看到博客。
